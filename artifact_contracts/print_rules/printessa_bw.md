# PRINTESSA_BW — Print Rules Contract (Non-Negotiable)

This document defines the default black-and-white print profile for all BSChapp artifacts.
All generators MUST comply. UI preferences do not override print rules.

---

## 1. Paper & Layout
- Paper size: US Letter (8.5 x 11)
- Orientation: Portrait
- Margins:
  - Top: 0.75 in
  - Bottom: 0.75 in
  - Left: 0.75 in
  - Right: 0.75 in
- Single-page preferred; spill to page 2 only if required

---

## 2. Typography
- Font family: Sans-serif (system-safe)
- Body text size: 12–14 pt
- Header text size: 16–18 pt
- Line spacing: 1.3–1.5
- No decorative fonts
- No italics for essential instructions

---

## 3. Color Rules
- Black text on white background ONLY
- No grayscale shading that may reproduce poorly
- Icons must be outline-style or text-based
- Color is never required to interpret content

---

## 4. Required Student Identification Block
Every artifact MUST include, at the top:

- Student Name: ______________________
- Date: ____________
- Class / Period: ____________

This block may not be removed or minimized.

---

## 5. Instruction Clarity
- Directions must be explicit and student-readable
- One task per section
- Avoid compound instructions
- Bulleted or numbered steps preferred

---

## 6. Accessibility & D75 Defaults
- Clear chunking of tasks
- Adequate white space between sections
- No dense text blocks
- Visual separation between questions
- Sentence frames allowed and encouraged

---

## 7. Forbidden Elements
- Background images
- Watermarks
- Hidden text
- Tracking codes
- Auto-generated identifiers tied to teacher identity

---

## 8. Override Rules
- This print profile overrides UI layout choices
- Teachers may select alternate profiles, but PRINTESSA_BW is the default
- Admin settings do not override print rules

---

## 9. Failure Behavior
If an artifact cannot be rendered under these rules:
- Generation must fail
- No partial output
- No silent fallback

Print integrity is non-negotiable.

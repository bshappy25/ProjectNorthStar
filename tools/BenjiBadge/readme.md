## Print Guidance

For best print quality:

* Print at 100% scale
* Use matte sticker paper for best gold tone
* Avoid JPEG conversion (PNG preserves transparency)

If exporting to JPEG:

* Add controlled background color first
* Then convert
* Never convert transparent PNG directly to JPG without background
